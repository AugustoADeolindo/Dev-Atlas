Como usar este material

Passo 1
Escolha um módulo

Comece pelo que te interessa ou siga a ordem sugerida de cima pra baixo. Não tem certo ou errado — o caminho é seu.

Passo 2
Leia e teste o código

Cada página tem exemplos práticos pra você copiar, colar e brincar. Mexer no código é a melhor forma de aprender.

Passo 3
Use o menu para navegar

Cada módulo tem um índice lateral com todos os tópicos organizados. Volte sempre que precisar revisar alguma coisa.
